# Enterprise network architecture specialist

An Enterprise network architecture specialist provides expert guidance on designing, evaluating, and improving enterprise network infrastructure. This includes defining trust boundaries, recommending architectures, identifying monitoring gaps, and documenting corrective actions with risk assessments.

## Who uses this

- **Network-architect**: Designs and evaluates enterprise network structures.
- **Network-engineer**: Implements and supports network infrastructure based on architectural guidance.
- **Network-operations-reviewer**: Assesses network operations for compliance and efficiency.
- **Managed-network-service-reviewer**: Reviews managed network services for alignment with enterprise standards.

## What it produces

- Network architecture recommendations with explicit trust boundaries and sequencing
- Corrective design guidance with risk ranking
- Troubleshooting runbooks with confidence and assumption registers
- Clear rejection/escalation language for adjacent specialty requests
- Monitoring and evidence gaps mapped to deployment readiness

## Validation

- 18 evaluation scenarios, pass rate 1
- Source coverage includes 14 documents from Cisco, IETF / RFC Editor
- Human verification and functional audit completed

## Limitations

- Requires detailed network topology and operational constraints from the user
- Requires access to monitoring data and deployment history for accurate readiness assessments
- Does not provide implementation-level coding or scripting
- Does not perform live network testing or packet analysis
- Does not replace on-site technical assessment or hardware evaluation
