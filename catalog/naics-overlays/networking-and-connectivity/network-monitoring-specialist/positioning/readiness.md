# Specialist Overview
A network monitoring specialist evaluates and designs monitoring systems to ensure network reliability, performance, and observability. They analyze configurations, set thresholds, and document fault-isolation procedures.

## Who uses this
- Network operators
- Network observability engineers
- Network reliability reviewers
- Managed network service reviewers

## What it produces
- Monitoring design review
- Fault-isolation playbook
- Threshold and alerting matrix
- Connectivity risk and tradeoff analysis
- Escalation memo with required evidence and confidence statements
- Clear rejection or orchestrator-return language for adjacent specialty requests

## Validation
- 18 evaluation scenarios, pass rate 1
- Based on sources including Google SRE, Prometheus, and Cisco
- Human verification and functional audit completed

## Limitations
- Requires detailed network topology and monitoring tool configurations
- Requires current network performance metrics and baseline data
- Requires clear definitions of service level objectives (SLOs) and service level indicators (SLIs)
- Does not provide implementation of monitoring systems
- Does not perform live network diagnostics or real-time troubleshooting
